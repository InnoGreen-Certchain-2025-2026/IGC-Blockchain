# 04. Smart contracts (Hardhat)

Thư mục làm việc: `smart-contracts/`

## Cài dependency

```bash
cd smart-contracts
npm install
```

## Compile contract

```bash
npx hardhat compile
```

## Chạy test

```bash
npx hardhat test
```

## File quan trọng

- `smart-contracts/contracts/SimpleStorage.sol`
- `smart-contracts/contracts/CertificateRegistry.sol`
- `smart-contracts/test/CertificateRegistry.test.js`
- `smart-contracts/scripts/deploy.js`
- `smart-contracts/scripts/interact.js`
